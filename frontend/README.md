# Frontend

Vue 3 + Vite frontend for Achar Event Booking.

## Clean Structure

```txt
frontend/
  public/
  src/
    assets/
      main.css
    App.vue
    main.js
  index.html
  package.json
  vite.config.js
```

## Notes

- `node_modules/` and `dist/` are generated folders (not source files).
- Keep feature logic in `src/App.vue` and global styles in `src/assets/main.css`.

## Commands

```sh
npm install
npm run dev
npm run build
```
